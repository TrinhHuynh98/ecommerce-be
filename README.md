# E-Commerce API with Node JS

## Installation

Download [nodejs](https://nodejs.org/en).

```bash
npm init
npm install express bcryptjs jsonwebtoken cors express-jwt body-parser morgan mongoose

# swagger
npm install swagger-ui-express swagger-jsdoc
```
